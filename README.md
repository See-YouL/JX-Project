# STM32学习笔记

## 说明

本库仅供交流学习使用

- 开发版: 野火霸道V2开发版
- 烧录设备: dap下载器
- 软件开发环境: Vscode(EIDE插件) + MDK
- 手册: 使用Doxygen生成的手册在项目文件夹下的Doc/html/index.html
